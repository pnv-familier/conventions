# This repository is for Familier project document

## Project installation
1. Create folder ```familier```
2. Open the terminal of the folder
3. Run these command:
```
git clone https://github.com/pnv-familier/conventions
git clone https://github.com/pnv-familier/mobile
git clone https://github.com/pnv-familier/backend
git clone https://github.com/pnv-familier/admin-hub
```
